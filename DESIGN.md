# 我要喝水吹头发 - 条码管理应用

## 项目简介
一个简单实用的条码管理应用，专为快速展示饮水机和吹风机条码设计。应用启动时自动调节屏幕亮度至最高，方便扫码使用。

## 核心功能

### 1. 条码展示页
- **全屏轮播**：左右滑动切换不同条码
- **自动亮度调节**：页面显示时自动将屏幕亮度调至最大
- **黑色背景**：突出条码显示效果
- **空状态引导**：无条码时自动引导用户添加

### 2. 设置管理页
- **图片上传**：支持相册选择或拍照上传条码图片
- **列表管理**：展示所有已保存的条码缩略图
- **删除功能**：快速删除不需要的条码
- **使用说明**：内置简单明了的使用指南

### 3. 底部导航
- Tab Bar 导航在条码展示页和设置页之间快速切换
- 紫色主题色，简洁美观

## 技术架构

### 技术栈
- **框架**：uni-app (Vue 3)
- **平台**：Android App
- **存储**：LocalStorage + Base64图片编码

### 数据结构
```javascript
// LocalStorage 存储格式
{
  barcodes: [
    {
      id: "timestamp_random",        // 唯一标识
      name: "饮水机条码",             // 可选名称
      imageData: "data:image/jpeg;base64,..." // Base64编码的图片
    }
  ]
}
```

### 核心API
- `uni.chooseImage()` - 图片选择
- `uni.getFileSystemManager().readFile()` - 文件读取，转Base64
- `uni.setScreenBrightness()` - 屏幕亮度控制
- `uni.getStorageSync()` / `uni.setStorageSync()` - 本地数据持久化

## 使用流程

1. **首次使用**
   - 打开应用，系统检测到没有条码
   - 自动跳转到设置页引导用户添加

2. **添加条码**
   - 点击"添加条码"按钮
   - 选择图片（相册或拍照）
   - 图片自动转换为Base64并保存到本地

3. **查看条码**
   - 切换到"条码"标签页
   - 屏幕亮度自动调至最高
   - 左右滑动切换不同条码

4. **管理条码**
   - 在设置页查看所有条码
   - 点击删除按钮移除不需要的条码

## 设计亮点

1. **极简操作**：功能单一，操作简单，老少皆宜
2. **自动亮度**：无需手动调节，打开即用
3. **本地存储**：无需网络，数据存储在本地，隐私安全
4. **快速切换**：轮播设计，多个条码快速切换
